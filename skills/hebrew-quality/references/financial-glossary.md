# Hebrew Financial Glossary — Authoritative Vocabulary

Use these terms consistently. When writing Hebrew financial analysis, always prefer the Hebrew term listed here over any English alternative or ad-hoc translation.

## Stock Market

| Hebrew | English | Notes |
|--------|---------|-------|
| מניה | Share / Stock | Use for individual shares; "נייר ערך" only for general securities |
| נסחרת ב- | Traded at | "המניה נסחרת ב-₪50" |
| שווי שוק | Market cap | |
| מחזור מסחר | Trading volume | |
| שיא כל הזמנים | All-time high | Never use "ATH" or "All-Time-High" |
| הנפקה | IPO / Issuance | |
| בורסה | Stock exchange | |
| החזקות ציבור | Free float | |
| שער המניה | Share price | |
| שער סגירה | Closing price | |
| מדד | Index | |

## Income Statement

| Hebrew | English | Notes |
|--------|---------|-------|
| הכנסות | Revenue | Not "מכירות" unless referring to specific sales line |
| עלות ההכנסות | Cost of revenue | |
| רווח גולמי | Gross profit | |
| שיעור רווח גולמי | Gross margin | Never "שוליים גולמיים" |
| רווח תפעולי | Operating profit / EBIT | |
| שיעור רווח תפעולי | Operating margin | |
| EBITDA | EBITDA | Keep English; optionally: "רווח תפעולי לפני פחת והפחתות" |
| רווח נקי | Net profit / Net income | Not "הכנסה נטו" |
| הפסד נקי | Net loss | |
| שיעור רווח נקי | Net margin | |
| רווח למניה | Earnings per share (EPS) | |
| הוצאות מו"פ | R&D expenses | |
| הוצאות הנהלה וכלליות | G&A expenses | |
| הוצאות מימון | Finance expenses | |

| Hebrew | English | Notes |
|--------|---------|-------|

## Common Verbs in Financial Context

| Hebrew | English | Usage |
|--------|---------|-------|
| רשמה | Recorded | "החברה רשמה הכנסות של..." |
| דיווחה | Reported | "החברה דיווחה על רווח נקי..." |
| הציגה | Presented | "החברה הציגה שיפור ב..." |
| נסחרת | Traded (f.) | "המניה נסחרת במחיר..." |
| גייסה | Raised | "החברה גייסה ₪50 מיליון..." |
| חילקה | Distributed | "החברה חילקה דיבידנד של..." |
| הונפקה | Was issued / IPO'd | "החברה הונפקה בשנת..." |
| צמחה | Grew (f.) | "החברה צמחה ב-15%..." |
| ירד/ירדה | Declined | Match gender to subject |
| עלה/עלתה | Increased | Match gender to subject |
| הסתכם/הסתכמה | Totaled | "הרווח הסתכם ב-..." |
