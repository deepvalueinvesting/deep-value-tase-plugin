# Hebrew Financial Formatting Reference

## Number Formatting

### Currency
- Shekel: ₪1,234,567
- Thousands: אלפי ₪
- Millions: מיליוני ₪ or מלש"ח (מיליון ש"ח)
- Billions: מיליארדי ₪ or מלרד"ש (מיליארד ש"ח)

### Percentages
- Always with % sign: 15.2%, -3.5%
- Basis points: נקודות בסיס (bps)
- Change notation: עלייה של 2.3%, ירידה של 1.1%

### Financial Ratios
- P/E: מכפיל רווח
- P/B: מכפיל הון
- EV/EBITDA: מכפיל EV/EBITDA
- ROE: תשואה להון
- ROA: תשואה לנכסים
- Current Ratio: יחס שוטף
- Quick Ratio: יחס מהיר
- D/E: יחס חוב להון
- Dividend Yield: תשואת דיבידנד

## Table Formatting

### Comparison Tables
Use markdown tables with Hebrew headers:
```
| מדד | חברה א' | חברה ב' | מנצחת |
|------|---------|---------|-------|
| מכפיל רווח | 12.5 | 15.3 | חברה א' |
```

### Financial Data Tables
```
| רבעון | הכנסות (מלש"ח) | רווח גולמי | שולי רווח |
|-------|----------------|-----------|----------|
| Q4 2025 | 1,234 | 456 | 37.0% |
```

## Chart Titles (Hebrew)

### Revenue & Profitability
- מגמת הכנסות רבעונית
- שולי רווח לאורך זמן
- הכנסות מול רווח תפעולי
- פילוח הכנסות לפי מגזר

### Balance Sheet
- הרכב מאזן
- מינוף פיננסי לאורך זמן
- הון חוזר רבעוני

### Valuation
- מכפילי תמחור — השוואה
- תשואת דיבידנד היסטורית

### Comparison
- השוואת הכנסות — [חברה 1] מול [חברה 2]
- השוואת שולי רווח
- השוואת מכפילי תמחור

## Disclaimer (Standard)
Always end with (plain text, no emojis, no borders):
```
ניתוח זה אינו מהווה ייעוץ השקעות. יש לבצע בדיקה עצמאית לפני קבלת החלטות השקעה.
```
