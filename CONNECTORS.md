# Connectors

## DeepValue MCP Server

This plugin connects to the DeepValue MCP server at `https://financials.deepvalue.co.il/mcp`.

The server provides 48+ financial data tools for the Israeli stock market (TASE), including:
- Company search and details with AI-generated insights
- Financial statements (income, balance sheet, cash flow)
- Market data (quotes, historical prices, intraday)
- Stock screening with 100+ metrics
- Maya (TASE regulatory) announcements
- Chart generation with Hebrew RTL support

All data tools require no authentication for read access.
